# 治験適格性スクリーニング (Clinical Trial Eligibility Screening)

治験の適格性をスクリーニングするためのWebベースの補助ツールです。
インタラクティブなフォームを通じて、患者が治験の適格基準（Inclusion Criteria）および除外基準（Exclusion Criteria）を満たしているかを判定のサポートを行います。

## 機能概要

以下の6つのステップで適格性を確認します：

1.  **基本適格要件**: 年齢、同意能力、診断、検体提出の可否など
2.  **疾患特性・全身状態**: RECIST測定可能病変、予後、ECOG PS、血圧など
3.  **臓器機能・検査値**: 血液、腎機能、肝機能、凝固系の検査値チェック（リアルタイム検証機能付き）
4.  **除外基準（病歴）**: 重複がん、手術歴、移植歴、アレルギーなど
5.  **除外基準（追加）**: 副作用の未回復、出血傾向、高血圧、感染症など
6.  **除外基準（治療・投薬）**: 消化管障害、併用薬、放射線療法、他の治験参加状況など

## 特徴

- **ステップ形式のUI**: 複雑な基準を段階的に確認できます。
- **リアルタイムバリデーション**: 検査値（ANC, 血小板, eGFRなど）を入力すると、基準値を満たしているか即座に判定します。
- **総合判定**: 全項目の入力後、適格/不適格/要確認の判定結果を表示します。

## 使用方法

1. リポジトリをダウンロードまたはクローンします。
2. `index.html` をWebブラウザで開いてください。
3. 画面の指示に従い、項目の入力・選択を行ってください。

## 免責事項

本ツールは治験適格性判断の**参考ツール**です。
最終的な適格性判断は、必ず治験実施計画書（プロトコール）に基づき、医師の判断によって行ってください。
